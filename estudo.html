DOCTYPE
Introdução, encontrada no topo dos documentos
html
Representa a raiz de um HTML
head
Contém informações gerais do documento, como titulo, imagens e código
body
Contéudo do HTML
h1
Representa o nível de título do HTML
p
Representa um parágrafo
style
Função utilizada para modificar o estilo do documento ou parte dele
script
Utilizado para referenciar um script
img
Utilizado para inserir imagens no HTML
br
Utilizado para quebrar linhas dentro do parágrafo
ul
Utilizado para fazer listas não ordenadas
ol
Utilizado para fazer listas ordenadas
li
Utilizado nas listas ordenadas e não ordenadas para marca-lás
